Home page banner slider

You can easly change slider effect from going this path:

magashop\js\common.js

and search below code

                	$('.flexslider').flexslider({
                        animation: "slide",
                        start: function(slider){
                          $('body').removeClass('loading');
                        }
                      });

Listed below are all of the options available to customize FlexSlider to suite your needs, along with their default values. For examples on how to use these properties for advanced setups, check out the Advanced Options section.

easing: "swing",               //{NEW} String: Determines the easing method used in jQuery transitions. jQuery easing plugin is supported!

Home Page Product Carousel

You can easly change carousel effect from going this path:

magashop\js\common.js

and search below code

                	$('#mix').jcarousel({
                        scroll: 1,
                        easing: 'swing',
                        animation: 750,
                        visible: 0,
                        auto: 0
                    });

Listed below are all of the options available to customize jcarousel to suite your needs, along with their default values. For examples on how to use these properties for advanced setups, check out the Advanced Options section.
